# G4 OS Docs

Documentação institucional e de produto do G4 OS em Mintlify.

## Rodando localmente

```bash
npx mint dev
```

## Estrutura inicial

- `docs.json`: configuração global do site
- `index.mdx`: página inicial
- `getting-started/`: paginas de onboarding
- `product/`: visão geral dos principais pilares do produto
