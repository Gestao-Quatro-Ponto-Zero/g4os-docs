## Para que servem as tags

Tags ajudam a organizar sessões com marcadores reutilizáveis e, quando necessário, metadados mais estruturados.

## O que você encontra aqui

- árvore de tags do workspace
- semântica visual, incluindo cores
- regras automáticas de aplicação por padrão de mensagem

## Casos comuns

- criar tags para áreas, projetos e prioridades
- ajustar a hierarquia para facilitar navegação
- usar autoaplicação para reduzir trabalho manual
- combinar tags com a visão de sessões para filtrar contexto

## Caminho no app

- `Configurações > Tags`

## Deep links úteis

- [Abrir Tags](g4os://settings/labels)
- [Abrir Sessões](g4os://allSessions)

## Dica prática

Se as tags estão sendo usadas só como lista plana, vale revisar a hierarquia e as regras automáticas. Isso normalmente melhora bastante a navegabilidade.
