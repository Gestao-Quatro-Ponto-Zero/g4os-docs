## O que é Cloud Sync

Cloud Sync é a superfície de backup e restauração do workspace.

Ele não substitui colaboração ao vivo. O foco aqui é preservar e recuperar dados do ambiente.

## O que você encontra aqui

- geração e gestão de token
- conexão e desconexão do sync
- status atual da sincronização
- push manual de dados locais
- preview de pull
- pull para restauração
- resultado da última operação

## Casos comuns

- configurar o sync em uma nova máquina
- enviar estado local para backup
- restaurar um workspace após recovery
- revisar se a conta ainda está conectada

## Caminho no app

- `Configurações > Cloud Sync`

## Deep links úteis

- [Abrir Cloud Sync](g4os://settings/sync)

## Importante

Use Cloud Sync para continuidade e recuperação de workspace. Para automação, fontes ou colaboração operacional, procure as áreas específicas do produto.
