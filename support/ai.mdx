## O que esta área controla

A tela de `IA` concentra os padrões globais de execução ligados a modelos e conexões com provedores.

## O que você encontra aqui

- conexão padrão usada pelo app
- modelo padrão associado à conexão
- nível de thinking padrão
- overrides por workspace
- lista de conexões configuradas
- ações como renomear conexão e revisar credenciais

## Casos comuns

- trocar o provedor padrão usado nas sessões
- mudar o modelo principal de trabalho
- ajustar o thinking padrão para equilibrar velocidade e profundidade
- revisar se uma conexão está pronta para uso
- corrigir credenciais ou reconfigurar uma conexão existente

## Como pensar esta página

`IA` define a base operacional do produto. Se o problema é “qual modelo ou qual provedor o G4 OS está usando”, normalmente a resposta está aqui.

## Caminho no app

- `Configurações > IA`

## Deep links úteis

- [Abrir IA](g4os://settings/ai)
- [Abrir Configurações do App](g4os://settings)

## Dica prática

Se um workspace precisa de comportamento diferente do padrão global, use os overrides por workspace em vez de alterar a base para toda a instalação.
