## O que esta área cobre

O fluxo de compartilhamento permite publicar uma sessão e manter o link sincronizado com o conteúdo mais recente.

## O que você consegue fazer

- publicar uma sessão online
- abrir o link publicado no navegador
- copiar o link
- atualizar o compartilhamento sem trocar a URL
- encerrar o compartilhamento e revogar o acesso
- importar uma sessão colaborativa

## Fluxo recomendado

1. Abra a sessão que deseja compartilhar.
2. Use o menu de compartilhamento no cabeçalho.
3. Publique, copie ou abra o link.
4. Quando necessário, atualize ou revogue o compartilhamento.

## Casos comuns

- demo assíncrona
- revisão de conversa por outra pessoa
- repasse de contexto para suporte ou operação
- envio de conversa publicada para stakeholders

## Caminho no app

- `Sessões > abrir uma sessão > menu de compartilhamento`

## Deep links úteis

- [Abrir Sessões](g4os://allSessions)

## Importante

Atualizar um compartilhamento publica a versão mais recente no mesmo link. Encerrar o compartilhamento remove o acesso externo àquela publicação.
