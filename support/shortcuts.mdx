## O que esta área cobre

`Configurações > Shortcuts` exibe a lista completa de atalhos de teclado disponíveis no G4 OS, organizados por categoria.

Os atalhos são gerados dinamicamente a partir do registro de ações do app — refletem sempre o estado atual do produto.

## Quando usar

- para descobrir atalhos de navegação entre sessões e painéis
- para verificar qual tecla executa uma ação específica no seu sistema operacional
- os atalhos variam levemente entre macOS (`⌘`) e Windows/Linux (`Ctrl`)

## Caminho no app

- `Configurações > Shortcuts`

## Deep links úteis

- [Abrir Atalhos](g4os://settings/shortcuts)
- [Abrir Input](g4os://settings/input)

## Dica prática

A tela de atalhos não tem configuração manual — para alterar a tecla de envio de mensagens, use [Input](g4os://settings/input).
