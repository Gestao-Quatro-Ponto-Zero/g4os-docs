## O que é esta seção

Esta seção documenta, com mais detalhe, as áreas que hoje aparecem na rota interna de `Support` do G4 OS.

Ela existe para ajudar quem está usando o produto a:

- entender onde cada configuração vive
- saber quando usar cada tela
- abrir mais rápido a área certa dentro do app

## Como usar

Para cada tema, você vai encontrar:

- o que aquela área controla
- tarefas comuns
- o caminho dentro do app
- quando relevante, um deep link `g4os://` para abrir a tela no desktop

> Os deep links `g4os://` funcionam quando o app desktop do G4 OS está instalado e registrado no sistema.

## Abrir a central no app

- Caminho no app: `Configurações > Support`
- Deep link: [Abrir Support no G4 OS](g4os://settings/support)
