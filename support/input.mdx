## O que esta área cobre

`Configurações > Input` controla como o campo de texto se comporta durante a digitação e no envio de mensagens.

## O que você encontra aqui

- **Auto Capitalisation** — capitaliza automaticamente o início de frases enquanto você digita
- **Spell Check** — sublinha palavras com possíveis erros ortográficos no campo de input
- **Send Message Key** — define qual tecla envia a mensagem: `Enter` ou `⌘ Enter` (macOS) / `Ctrl+Enter` (Windows/Linux)

## Casos comuns

- desativar auto capitalização ao digitar comandos ou código
- ativar spell check para mensagens mais longas e formais
- trocar a tecla de envio para `⌘ Enter` quando você quer poder quebrar linhas com `Enter`

## Caminho no app

- `Configurações > Input`
