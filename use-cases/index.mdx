## Explore o G4 OS com exemplos prontos

Esta seção reúne casos de uso com datasets de exemplo e sessões pré-configuradas. Cada caso mostra como usar o G4 OS para um tipo específico de análise ou operação — basta clicar no deep link e o app abre já com o contexto configurado.

Cada caso inclui:

- um **dataset de exemplo** pronto para download
- um **deep link que abre uma sessão no G4 OS** com o contexto e instruções já prontos
- descrição do que está no dataset e que análises são possíveis

## Como funciona o deep link

Cada página de caso de uso tem um botão **"Abrir no G4 OS"**. Ao clicar:

1. O G4 OS abre uma sessão nova no modo Execute
2. A mensagem inicial já está pré-preenchida com contexto e instruções
3. O assistente executa a análise automaticamente — baixa arquivos, roda código, gera gráficos

Para isso funcionar, o app desktop do G4 OS precisa estar instalado e em execução.
