## Como funciona o deep link

Cada página tem um botão **"Abrir no G4 OS"**. Ao clicar, o app abre uma sessão no modo Execute com o contexto e as instruções já prontos — sem copiar prompt, sem configurar nada.

Para isso funcionar, o app desktop do G4 OS precisa estar instalado e em execução.
