## Aprenda com dados reais (anonimizados)

Esta seção reúne casos de uso construídos a partir de dados reais de uma empresa de serviços B2B, totalmente anonimizados e transformados em datasets sintéticos para uso educativo.

Cada caso inclui:

- um **dataset `.xlsx` pronto para download**
- um **deep link que abre uma sessão no G4 OS** já com o contexto e a análise configurados
- descrição do que está no dataset e que análises são possíveis

## Como funciona o deep link

Cada página de caso de uso tem um botão **"Abrir sessão no G4 OS"**. Ao clicar:

1. O G4 OS abre uma sessão nova
2. A mensagem inicial já está pré-preenchida com contexto e instruções
3. O assistente baixa o dataset diretamente do GitHub e inicia a análise

Para isso funcionar, o app desktop do G4 OS precisa estar instalado e em execução.

## Sobre os datasets

Os datasets foram gerados a partir de dados reais com as seguintes transformações:

- identidades substituídas (clientes, vendedores, produtos)
- valores ajustados por fator de escala para descaracterização
- estrutura e correlações preservadas para manter coerência analítica
- empresa fictícia: **Nexus Consultoria** — serviços B2B genéricos

Os arquivos ficam hospedados neste repositório em `use-cases/datasets/` e podem ser baixados diretamente ou usados via deep link.
