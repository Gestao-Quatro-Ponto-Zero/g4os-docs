## O que este caso de uso cobre

O workflow **G4 OS Design** transforma descrições em texto em entregas de design de alta fidelidade — slides 1920×1080, mockups de app, infográficos e animações com export MP4/GIF. Tudo gerado como HTML no G4 OS, sem ferramentas externas.

## Antes de começar: instalar o workflow

O workflow **G4 OS Design** precisa estar instalado no seu workspace. Ele está disponível no Marketplace do G4 OS.

<Steps>
  <Step title="Abrir o Marketplace">
    Acesse o Marketplace pelo menu lateral do G4 OS ou pelo deep link abaixo.

    [Abrir Marketplace](g4os://marketplace)
  </Step>
  <Step title="Buscar e instalar">
    Pesquise por **"G4 OS Design"** e instale no seu workspace. Após a instalação, o workflow fica disponível como `/g4os-design` em qualquer sessão.
  </Step>
  <Step title="Confirmar a instalação">
    Abra uma sessão nova e digite `/g4os-design` — o G4 OS vai pedir seu briefing e começar a criar.
  </Step>
</Steps>

O G4 OS vai perguntar sobre tema, estilo visual, quantidade de slides e conteúdo — e depois entrega o deck completo como HTML pronto para visualizar, exportar em PDF ou converter para PPTX.

## O que o workflow entrega

| Tipo | Formato de entrega | Export |
|------|--------------------|--------|
| Slides / deck | HTML 1920×1080 por slide | PDF, PPTX |
| Protótipo de app | HTML interativo com frames nativos | — |
| Animação / motion | HTML com timeline CSS/JS | MP4, GIF |
| Infográfico | HTML estático, tipografia premium | PNG, PDF |
| Variações de design | 3 direções lado a lado com live tweaks | — |

Exemplos de briefings que funcionam bem:

- `"Crie um deck de 8 slides para uma proposta comercial de software B2B, tom executivo, tema escuro"`

## Análises e entregas possíveis

- **Decks executivos** — propostas, pitches, relatórios mensais

- **Protótipos de produto** — mockups clicáveis para validação de UX
- **Materiais de marketing** — landing pages estáticas, banners, infográficos
- **Motion design** — animações para abertura de vídeos ou apresentações
